learn Sass

9月 02 2018


编译SASS

  1. 单文件转换: sass input.scss output.css

  2. 单文件监听: sass --watch input.scss:output.css

  3. 目录监听：sass --watch dir:dir

  4. 局部scss文件：_filename，以下划线开头，sass会忽略这些文件，除非我们明确调用他们。

  5. sass导入局部scss文件：@import 'filename'，可以省略下划线和后缀scss。

SASS变量

  1. $var: val，一次赋值，多次使用。

  2. 为了方便其他sass文件共用变量，可以创建一个scss文件，专门用于存放变量，比如：_variable.scss。为了使用变量，必须在其他sass文件之前加载变量sass文件。

  3. 颜色变量通常使用方法：首先定义一组变量，得到一组调色板，根据具体用途，把变量再赋值给另一个变量。类似的有，字体栈。

  4. 变量如果是相同单位可以进行简单的计算,加减时要使用相同的度量单位。

嵌套选择符

  1. 用花括号把子元素放在父元素中，这就是SASS的嵌套。

  2. 引用父辈选择符：&，使用“&”代表父辈选择符。

  3. 多层嵌套，sass是程序，支持多层嵌套，只需要在子元素中再包含它的子元素。

继承或扩展属性

  1. 继承样式: @extends selector

  2. 占位选择符：%name，表示只能用作其他样式基底的样式。解决继承功能太霸道的问题。

  3. 只扩展相关的元素。

混入

  1. 混入指代一组想重复多次使用的CSS属性，可重用。

  2. 创建混入：@mixin name{}，在花括号中添加样式代码。

  3. 使用混入：@include name

  4. 混入函数：@mixin name($var) {}，在花括号中添加样式代码。

  5. 混入库，比如neat，bourbon，compass（不仅仅是混入库，包含其他很多功能），susy。

  6. 插值（interpolation）：#{$var}

  7. 可选参数：@mixin name($var1, $var2: null, $var3: null)，这里var2和var3都是可选的，当我们不提供他们的值的时候，sass会忽略这些值。

  8. 参数默认值：@mixin name($var1, $var2: 1.2em, $var3: red)，这里var2和var3都指定了默认值，当我们省略他们不写的时候，会使用这些默认值，否则会覆盖默认值。

  9. 明确传值形参：@include name(2em, $color: red)
