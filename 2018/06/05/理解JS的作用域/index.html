作用域

函数作用域 function scope

  • 函数作用域是指在函数内声明的所有变量在函数体内始终都是可见的。这意味着变量在声明之前就是可用的了。这就产生了声明提前hoisting。

全局作用域 global scope

  • 全局作用域是指在全局作用域内声明的变量在JS代码中的任何地方都是可见的。

  • 给一个未声明var的变量赋值，JS会把该变量作为全局对象的属性使用。

块级作用域

  • ES6中可以使用let，const创建一个块级作用域，简单理解块级作用域指的就是花括号内的每一段代码都具有各自的作用域。

作用域链

  • 作用域链是一个对象列或链表（变量对象），这组对象定义了这段代码“作用域中”的变量。
  • 作用域的用途是保证执行环境有权访问的所有变量和函数的有序访问。
  • 作用于的前端始终都是当前执行的代码所在环境的变量对象。作用域链中的下一个变量对象来自包含外部环境，如此类推下去。

变量

变量类型

  • 三种不同的数据类型：基本类型、引用类型、Symbol类型。
  • 基本类型： null、undefined、Number、String、Boolean。按值访问，可以操作保存在变量中实际的值。
  • 引用类型：Object。按引用访问。

复制变量值

  • 基本类型：在变量对象上创建一个新值，然后把该值赋值到新变量分配的位置上。
  • 引用类型：同样在变量对象上创建一个新值，但这个值是一个指针，指向存储在内存中的对象。这意味着两个变量实际上引用同一个对象，改变其中一个变量就会影响另一个变量。
