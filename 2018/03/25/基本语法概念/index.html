JS基础概念

3月 25 2018

基本语法概念

  • ES采用驼峰大小写格式，第一个字母小写，剩下的单词首字母大写
  • ES5引入了严格模式的概念，为JS定义了一种不同的解析和执行模型，在严格模式下，一些不确
    定的行为得到处理，一些不安全的操作也会抛出错误。使用方法：添加代码use strict,告诉JS
    引擎切换到严格模式。

  • 始终在控制语句中使用代码块，并且加上分号。

  • 关键字是具有特定用途的关键字；保留字暂时还没有特定用途，但将来可能被用作关键字。

  • 声明了变量但未赋值的变量会有一个特殊值——undefined。
  • 省略了var操作符会创建一个全局变量。
  • 尽量减少在局部作用域中定义全局变量。
  • 5种简单数据类型：Undefined，Null，Boolean，Number，String；一种复杂数据
    类型：object。
  • 使用typeof操作符检测变量的数据类型，typeof是一种操作符而不是函数。
  • 函数本质上时对象，但用typeof操作符检测时，显示的是function。
  • 对尚未声明的变量只能执行typeof操作，否则出错，这是跟undefined区别

  • 对声明了但未赋值的（未初始化）的变量和未声明的变量执行typeof操作符都会返回
    undefined。

  • null表示一个空对象指针，使用typeof会返回object。

  • 将一个新定义的变量赋值为null的好处是直接检查null值就可以知道相应变量是否已经保存
    了一个对象的引用。
  • undefined表示变量声明了但没有值，null是声明了并且值为null。

  • null和undefined两者使用相等操作符返回true。

  • 可以使用Boolean()转型函数将其他值值转换为Boolean类型。

  • 其他类型转换为Boolean类型的规则：

    如下表格：

数据类型 true false
Boolean true false
String 非空字符串 空字符串
Number 任何非零数字值包括无穷大小 0和NaN
Object 任何对象 null
Undefined n/a not applicable 不适用 undefined

  • 整数可以通过十进制,八进制,十六进制表示.其中八进制的第一位必须是0,后面的数字不能超过7
    ;十六进制则是0x后跟0~9和A~F.

  • 不同进制相加自动转换为十进制再进行运算.

  • ES保存浮点数值所需要的内存空间是保存整数值的两倍,所以ES会自动把某些浮点数值自动转换
    为整数值,比如:1.和10.0都会自动转换为整数.
  • 科学计数法(e表示法):e前面的数值乘以10的指数次幂.比如:3.12e2=3.121010=312
  • 0.1+0.2==0.3//false，浮点数的最高精度是17位数，这是基于规范数值的浮点计算的通病。
    因此，不要做测试某个特定的浮点数值.
  • ES中保存的最小数值是Number.MIN_VALUE，能够保存的最大数值Number.MAX_VALUE，超过这
    个范围则会转换为特殊的Infinity值。
  • 可以使用isFinite()函数来检测是否在数值范围中，是则返回true；
  • NaN表示非数值Not a Number 是一个特殊的数值。有两个特点：任何涉及对NaN操作都会返回NaN
    ，NaN与任何值都不相等，包括NaN本身。NaN==NaN; //false
  • 可以使用isNaN()来判断一个值是否为非数值。
  • 非数值转换为数值：Number(),parseInt(),parseFloat,Number()可用于任何数据类型，另
    两个函数则用于特定字符串转换为数值。Number()转换规则:

    如下表格：

类型 转换值
Boolean 真假值分别转换1和0
Number 直接返回
null 0
字符串 数字=》十进制数值，忽略前导零；有效的浮点格式=》浮点数值；有效的十六进制=》相同大小的十进制；空字符串=》0；除上述以外=》 NaN
  • 相对于Number(),parseInt()显得更强大：可以转换八进制，十六进制，只需要指定多一个进制参
    数；可以识别含有可解析为整数的字符串，例如：’123asdf’=》123；
  • 相对于parseInt，parseFloat可以解析有效的小数点；将会忽略所有的前导零；没有指定进制的
    参数
    ###String

  • 字符串是不可变的，一旦创建，他们的值就不能改变；想要改变变量中保存的字符串则需要销毁原
    来的字符串，再使用一个新值字符串填充该变量。

  • 转换为字符串：

    1. toString，除了null和undefined都有这个方法，返回相应值得字符串表现，特点是可以添加一个输出数值的进制，实现进制之间的相同转换。
    2. String()转型函数，将所有类型的值都转换为字符串显示。null=》”null”,undefined同理。
      否则直接调用toString();
    3. hack: 使用加号操作符与一个空字符串加在一起。

###object

  • Object类型是所有对象的基础，它所具有的任何属性和方法也存在于更具体的对象中。
  • 具有的属性：constructor，构造函数。
  • 具有的方法：hashOwnProperty(“[“)，isPrototypeOf(Object),propertyIsEnumerable(),
    toString,valueOf();
