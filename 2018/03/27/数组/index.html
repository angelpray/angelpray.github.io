JS数据结构——数组

3月 27 2018

数组

  • 数组是最简单的内存数据结构。

  • 大多的语言的数组存储的是同一类型的值，在JS中，同一个数组中可以保存不同类型的值，但尽量不要这么做。

创建和初始化数组

  • 创建数组的方式：new Array(),new Array(length),new Array(element1,element2,...),[],[element1,element2,...]

  • 在js中，数组是一个可以修改的对象，可以随意添加删除元素，动态增长或减短。

添加/删除/替代元素

  • 添加元素：Array[Array.length]=element,Array.push(),Array.unshift;前者是添加元素到最靠后的元素，后者则相反。

  • 删除元素：pop(),shift();前者是删除最靠后的元素，后者则相反。

  • 任意位置添加/删除/替代元素：splice(),第一个参数表示要删除或者插入的元素的索引值；
    第二个参数是删除元素的个数，第三个参数往后是要添加到数组的值。

数组合并

迭代器函数

  • every(function)迭代数组中的每个元素，直到返回false；

  • some(function)迭代数组中的每个元素，直到返回true；

  • forEach(function)迭代整个数组，执行某个函数，没有返回值；

  • map(function),filter(function);前者保存的是元素执行函数之后的所有结果，后者则是只保存返回true组成的元素值。

  • reduce(function)，稍微复杂一点的是接收的函数有四个参数：previousValue,currentValue,index,array,其中previousValue表示的是上一次运算的和；
    reduce方法停止执行后返回这个累加器。一般用于数组中的所有元素求和。return previousValue+currentValue

ES6/7新增的数组功能

  • 使用forEach和箭头函数进行数组迭代：

  • 使用for…of循环迭代

  • 使用ES6的迭代器@@iterator,这个属性需要Symbol.iterator来访问。

  • 除了上述获取iterator的方法外还增加其他方式：entries，keys，values。

  • entries返回一个含有键值对的迭代器对象

  • keys则返回只含有键的迭代器对象

  • values返回只含有值的迭代器对象，跟Symbol.iterator属性一样。

  • 目前有点疑惑的是：使用for…of居然可以直接输出整个迭代器对象的值，而不用next().value如此一个个进行下去。因为next()返回的是一个对象，需要输出value属性，而
    for…of确实可以直接输出迭代器对象的值。

  • Array.from(array,function)，根据已有的数组创建一个新数组。

  • Array.of(element1,element2,…)，根据传入的参数创建一个新数组,相当于[element1..]
    指定

  • fill(element,index)，用静态值填充数组。创建数组并初始化值:let arr=Array(6).fill(1)

  • copyWithin(pasteIndex,copyIndex),复制数组中的一段元素到统一数组指定的位置。copyArray = [1, 2, 3, 4, 5, 6]; copyArray.copyWithin(1, 3, 5);//[1,4,5,4,5,6]
