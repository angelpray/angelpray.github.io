ES6永不停息(十一)——Set和WeakSet

5月 28 2018

Set

  • ES6提供的新的数据结构Set，类似于数组，成员的值都是唯一的，没有重复的值。

  • Set(具有iterator接口的其他数据结构)作为参数，用来初始化。

  • Set加入值得时候不会发生类型转换。判断两个值是否不同的方式类似全等运算符，区别：全等
    运算符会认为NaN不等于自身，而Set会认为NaN认为NaN等于自身。另外，对象总是不相等的。

Set实例的属性和方法

  • Set属性：
  1. Set.prototype.constructor：构造函数
  2. Set.prototype.size:返回Set实例的成员总数
  • Set方法：
  • 操作方法：
  1. add(value) return Set
  2. delete(value) return Boolean
  3. has(value) return Boolean
  4. clear() no return
  • 遍历方法：
  1. keys() return key
  2. values() return value
  3. entries() return [key,value]
  4. forEach() no return

WeakSet

  • 跟Set类似都是不重复的值集合，与Set有两个区别
  1. WeakSet的成员只能是对象，不能是其他类型的值。
  2. WeakSet中的对象都是弱引用，当其他对象不再引用WeakSet成员对象，垃圾回收机制会自动回
    收该对象所占的内存。因此，它不可遍历。

  • 作用：所引用的对象的其他引用被清除了，则该WeakSet的成员会被自动清除。

  • WeakSet方法：

  1. add(value) return WS
  2. delete(value) return BL
  3. has(value) return BL
