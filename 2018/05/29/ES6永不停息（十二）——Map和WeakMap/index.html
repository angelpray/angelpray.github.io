ES6永不停息(十二)——Map和WeakMap

5月 29 2018

Map

  • ES5以前只能用字符串当做键（Symbol除外）。Map的出现解决了这个问题，键的范围不限于字符串。
    Map结构提供了值-值的对应。值可以是任意类型。

  • new Map(实现了iterable接口且每个成员都是双元素的数组的数据结构)

Map实例的属性和操作方法

  • 属性：
  1. size属性，返回Map结构的成员总数。
  • 操作方法：
  1. set(key, value)，设置键名key对应的键值value，然后返回整个Map结构。
  2. get(key),get方法读取key对应的键值，如果找不到key，返回undefined。
  3. has(key)， 返回一个布尔值，表示某个键是否在当前Map对象中。
  4. delete(key),删除某个键，返回BL
  5. clear，清除所有成员，没有返回值。
  • 遍历方法
  1. keys
  2. values
  3. entries
  4. forEach

WeakMap

  • 类似Map，区别两点：
  1. 只接受对象作为键名，null除外。
  2. WeakMap键名所指的对象，不计入垃圾回收机制。
  • 作用：所引用的对象的其他引用被清除了，则该WeakSet的成员会被自动清除。
  • API上的区别：
  1. 没有遍历方法，没有size属性。不支持clear方法。
