bind,call,apply【搞个明明白白系列】

5月 18 2018

bind()

  • 根据MDN
  • 语法：func.bind(thisArg[,arg1[,arg2[, ...]]])
  • 参数thisArg是作为原函数运行的this指向
  • arg1,arg2…，这些参数则是在实参之前传递给绑定的方法。

  • 返回值：由指定的this值和初始化参数改造的原函数拷贝。

  • 理解：bind()会创建一个新函数，也就是绑定函数。新函数和被调函数具有相同的函数体。当新
    函数被调用时，this值就绑定到bind的第一个参数！例子：

  • 应用：

  1. 创建一个函数，使这个函数有一个相同的this值
  2. 偏函数
  3. 配合setTimeout，因为使用setTimeout方法的时候，this关键字会变成指向window。需要显
    式把this绑定到回调函数。
  4. 快捷调用，为一个需要特定的this值得函数创建一个捷径（shortcut）

call()

  • 根据MDN
  • 语法： func.call(thisArg,arg1,arg2,...)
  • 参数thisArg是作为函数运行时指定的this值。(非严格模式下，null和undefined会自动替换成全局对象window)
  • 当绑定函数被调用时，这些参数将置于实参之前传递给绑定的方法。
  • 返回值： 返回值是调用的方法的返回值，如果方法没有返回值，则返回的是undefined。
  • 理解：方法调用一个函数，其具有一个指定的this值和分别提供的参数（参数列表）

  • 一个用call()实现继承的例子：写一个方法，然后让另一个对象来继承它。

  • 其他应用：

  1. 使用call()调用匿名函数
  2. 使用call()调用函数并且指定上下文的’this’

apply()

  • 前面说了call，事实上，apply和call作用是一样的，都是调用一个函数，其具有一个指定
    指定的this值。

  • 唯一的区别：apply方法接收的是数组（类数组也行，比如arguments）作为参数，而
    **call则是接收若干个参数的列表。

  • 应用

  1. 使用apply()结合内置函数
    ```JavaScript
    var numbers = [2,7,9,6,10];
    var max = Math.max.apply(null, numbers);
    var min = Math.min.apply(null, numbers);

总结：

  1. call,apply都是直接执行调用他们的函数，返回结果。
  2. bind返回的绑定了this值之后的原函数的拷贝。
  3. 他们都是在函数的原型中的方法。
