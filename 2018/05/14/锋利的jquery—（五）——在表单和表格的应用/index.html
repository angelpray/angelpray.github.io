锋利的jquery(五)——在表单和表格中的应用

5月 14 2018

表单

  • IE6不支持除了超链接元素之外的：hover伪类选择器，使用jquery来弥补。

  • height()只会返回内容的宽度而不会管有没有设置box-sizing。跟css(“height”)的方法区别在于，
    前者返回的是纯数字，后者则是会返回一个带单位的字符串。

  • attr和prop

  1. 对象的属性值properties和HTML中的属性attribute
  2. value值内容
    • attr反映的是HTML源码value属性的初始文本内容
    • prop反映的是input当前的文本内容
    • val也是返回当前内容
    • defaulValue是JSDOM对象属性，也是初始文本内容
  3. 纯粹和不纯粹映射
    • 这个取决于属性类型，比如id可以随意定义，则prop是对attr的纯粹映射
    • 但有些属性类型是有合法类型的，如type的合法类型有text,password，这时候如果设置
      type="hello"，则attr返回hello，prop返回的是text。
      4.

  • 表单元素有个特点：刷新网页后，其值会保持不变。当需要在表单元素做筛选的时候，需要在DOM
    刚加载完立刻触发事件。这里浏览器有一些差异，某些浏览器刷新网页后并不保存表单元素值，
    比如chrome。ff和ie则保存。

  • keypress和keydown，keyup最大的区别在于：keypress是插入了字符才触发，而后两者在松开和按下就立刻触发无论是否插入了字符。
