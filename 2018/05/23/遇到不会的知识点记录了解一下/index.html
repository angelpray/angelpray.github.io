    Array.prototype.reduce(callback,initialValue)

  • callback有四个参数：accumulator，currentValue，currentIndex，array；
  • initialValue：用来第一个调用callback的第一个参数的值，有两种情况：
  1. 当没有提供initialValue的时候：accumulator使用数组的第一个元素，currentValue则从第二项开始。
  2. 当提供了initialValue的时候，accumulator就是initialValue，并且currentValue是从数组第一项开始。

dev Dependencies 和dependencies的区别

  • npm install 有两种命令可以把需要安装的模块写入到package.json文件中：--save-dev以及
    --save,在package.json中的区别就是：前者会被写入到devDependencies对象中，后者则会被
    被写入到dependencies对象中。
  • 那么package.json中，这两个对象到底有何区别呢？
  • 经过查阅资料，发现dev依赖里面的插件是只用于开发环境，不用生产环境！，而dependencies
    是需要发布到生产环境中的。
  • 总结：依赖是程序正常运行需要的包。dev依赖则是开发过程需要的包，比如单元测试的包之类的。
    node的package.json中就包含这两个对象。
