SQL增删改查

（连接mysql mysql -u root -p）

删 DELETE

DELETE FROM table WHERE condition

增 INSERT

INSERT INTO table(key) VALUES(value)

改 UPDATE

UPDATE table SET key = value WHERE condition

查 SELECT

SELECT * FROM table WHERE

子句

SQL语句的一部分

WHERE条件子句

WHERE name=’zjr’ AND score>99
WHERE name=’zjr’ OR name=’zdx’

ORDER排序子句

ORDER BY age ASC/DESC
ASC:ascend，什序
DESC：descend，降序

多条件排序
ORDER BY price ASC，age DESC

GROUP聚类子句-合并相同

统计班级人数
SELECT class,COUNT(class) FROM student_table GROUP BY class;
COUNT函数可以计数。

分页

  1. 所有数据给前台
  2. 后台只给一部分数据给前端
