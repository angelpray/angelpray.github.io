响应式web设计

10月 19 2018


响应式web设计

  1. max-width: 100%和width:100%的区别，max-width会考虑自身的宽度，只有容器比图片宽度笑才会缩放占满可用空间，而width无论什么情况下都会占满可用空间。

  2. 断点：就是某个宽度临界点，跨过这个点布局就会发生显著变化。断点应该由内容和设计本身决定而不是那些流行的设备！

  3. 媒体查询：根据设备的能力应用特定的css样式。

  4. 规范批准流程：WD【working draft】,CR【candidate recommendation 候选推荐】,PR【proposed recommendation 建议推荐】,REC【recommendation 推荐标准】.

  5. 在link标签中使用媒体查询，media=”screen and (min-width=550px)”,通过and将多个媒体查询串在一起。

  6. 在@import使用媒体查询，@import url(style.css) screen and (max-width:550px);使用@import会增加HTTP请求英雄加载速度。

  7. 在CSS中使用媒体查询，@media screen and (max-device-width: 400px){style},默认是针对所有设备，因此可以省略screen。

  8. 媒体查询可以测试的特性：width（最常用了）、height、device-width（设备屏幕宽度，废弃）、device-height（设备屏幕高度）、orientation（设备方向）、aspect-ratio（视口宽高比）、resolution（分辨路）。都可以加上min和max前缀指定范围。

  9. 所有的CSS文件都会被下载下来，只是下载下来不必立即应用，优先级比较低而已。

  10. 阻塞渲染是指该资源是否会暂停浏览器的首次页面渲染。

  11. 视口的meta标签：<meta name="viewport" content="initial-scale=1.0,width=device-width">

  12. 区块转换成弹性比例大小：目标/上下文。

Flexbox布局

  1. Flexbox，有的需要前缀。自动加前缀，autoprefixer。

  2. 魔法属性，如果指定了flex布局，margin-left,margin-right指定为auto值的时，它让最后一项用上改侧所有可用的外边距。

  3. flex-direction,flex-wrap,flex-flow

  4. inline-flex，行内伸缩，比如a标签，可以在行内进行伸缩而不用变成块级。

  5. Flexbox的方向：align-items、align-self（可以覆盖align-items）、justify-content、

  6. flex的默认值：0 1 auto；表示宽度由里面的内容自动确定。

  7. 使用列+min-height=100%（HTML,BODY）可以实现页脚粘附。

  8. 把主内容放在页面的开头是个聪明的做法,结合使用order，甚至可以模仿所谓的“圣杯布局”

响应式图片

  1. 根据分辨率切换不同分辨率的图片:通过srcset属性+size属性联合切换。（少见到，忽略）

  2. 根据不同的视口大小使用不同的图片：picture元素，picture元素只是一个容器，需要使用source(srcset)配合img使用。在source中可以使用media媒体查询属性。

  3. 图片格式的替换，type + srcset，结合使用。type指明优先使用的图片格式image/后缀

HTML5与响应式设计

  1. 可以把多个元素放到a标签里

  2. 腻子脚本：用腻子可以填补墙上的坑洼不平来比喻填补老版本浏览器的功能缺失。

  3. HTML5响应式视频：删除width属性，添加 max-width:100%;height:auto;

  4. 学习了添加B站的视频

  5. 离线应用，localstorage，service workers

CSS3新特性

  1. 响应式多列布局：column-count、cloumn-gap、column-rule、column-width

  2. 断字：word-wrap: break-word;

  3. 截断文本：text-overflow[ellipsis],width,white-space;

  4. CSS分支,特性查询 @support (attrub: value) ，使用 Modernizr进行特性检测。

  5. 属性选择符，属性是当做一个字符串。比如 src=”1.jpg 2.jpg”,开头是1.jpg,不能以2.jpg开头去查找属性。

  6. CSS calc(),可以使用CSS calc方法进行简单的计算。比如计算父元素的宽度的一半减去10像素。.thing{width: calc(50%-10px)}

  7. CSS4选择器，工作草案，只是了解即可。比如：a:has(p){width:100px;},这个规则可以给一个包含p段落的a标签设置宽度是100px.

  8. 相对视口的长度【相对单位】： vw vh vmin vmax vmax。 可以使用这些相对单位。
    vw视口宽度， 1vw = 1% 视口宽度，
    vh视口高度， 1vh = 1% 视口高度
    vmin视口中的最小值，等于vw或vh中较小的值的1%
    vmax视口中的最大值，等于vm或vh中最大的值的1%

  9. HSL, 色相，饱和度，亮度 hue saturation lightless

  10. HSLA RGBA 不透明度 ， 0是透明 1是不透明

  11. opacity是不同明度，他与上面的区别在于：opacity会影响整个元素，而上面的只会影响元素特定的方面，比如背景。

  12. css计数器

  13. var(),CSS自定义属性和变量

CSS3高级技术

  1. 文字阴影 text-shadow: x y 模糊程度 颜色【通过逗号分隔，可以添加多个阴影效果】

  2. 盒阴影 box-shadow: 内阴影 x y 模糊程度 阴影尺寸 颜色【同理，通过逗号分隔，可以添加多重阴影效果】

  3. 背景渐变：background: linear-gradient(方向，颜色1，颜色2)，为了兼容不支持渐变效果的浏览器，可以定义一个背景颜色。

  4. 色标：定义颜色什么时候开始和结束，用于把渐变中的某个点设定为特定颜色。颜色 颜色位置

  5. 通过高分辨率的媒体查询加载高分辨率背景图片。

  6. CSS滤镜：
    drop-shadow【跟box-shadow有相同的参数，drop-shadow可以给元素或图片非透明区域添加投影】
    blue【模糊背景，px】
    brightness【高亮，全黑0-1正常，大于1则是更亮】
    contrast【对比度，跟上同理】
    grayscale【灰度化，跟上同理，但不能超过1，1表示完全灰度化】
    invert【元素反色，跟上同理】
    opacity【不透明度，跟上同理】
    ..CCS滤镜是基于SVG滤镜

  7. 可以同时使用多个滤镜，不同的滤镜只需要用空格分开

SVG与响应式设计

  1. 矢量图是使用相对点来保存数据，可以缩放到任意大小而不会损失清晰度。

  2. 视口：描述在设备上能够观看内容的面积。

  3. 视框：定义了SVG中所有形状都需要遵循的坐标系。viewbox = x轴的位置 y轴的位置 坐标系的宽度 坐标系的高度

  4. title, desc提高SVG文档的可读性

  5. defs标签，用于存储所有可以复用的元素定义的地方，如梯度，符号、路径等

  6. g元素，把其他元素组合起来

  7. SVG形状，有现成形状： path rect circle ellipse line polyline polygon

  8. SVG路径，由任意数量的连接点组成的，可以自由创造你想要的形状。

  9. 使用图像编辑器生成SVG。

  10. 不需要自己完全做出一个SVG，网络很多资源。

在web页面中插入SVG

  1. img标签，SVG能做的和其他插入的图片差不多。

  2. object标签，object标签是W3C推荐用于装载非HTML内容的容器。可以使用这个标签插入SVG。

type是内容弄的mime类型。采用这种插入方式可以被JavaScript访问。而且能在浏览器不支持的时候做出简单的反馈。

  1. background-image：url(“img/filename.svg”);如果对SVG的需要只是静态背景图片或者图标，那么尽量通过背景图像的方式插入SVG文件。

  2. data URI，统一资源标识符，用于引入外部资源，如图像。

  3. base64编码：base64是一种基于64个可打印字符来表示二进制数据的表示方法。
